от BIONIC University

BIONIC University в рамках развития концепции практического обучения запустил собственный проект по разработке програмного обеспечения - BU Dev Studio. Участники проекта работают над продуктом в команде под руководством опытных тим-лидеров, таким образом, изучая на практике весь цикл разработки продукта.

Участие в проектах предполагает наличие у кандидата высокого уровня подготовки и будет доступно только выпускникам BIONIC School. Срок разработки бизнес-решения: 10-15 недель. Это вовлекает команду в активное взаимодействие с заказчиком: демо, релизы, обратная связь. Что нужно знать кандидату?

Кандидат должен обладать базовыми знаниями по процедурному и объектно-ориентированному программированию (понятие переменная / постоянная, операторы / операции, массив и т.д.), а также основ алгоритмизации. Предварительных знаний Java не требуется. Наличие сертификата о завершении любой из программ BIONIC School, BIONIC University и BIONIC University Pro гарантирует 10% скидки на обучение.

Что будет знать и уметь выпускник?

Закончив программу, выпускники будут владеть базовыми знаниями, необходимыми для эффективного изучения платформы Java Standard Edition, ознакомятся с правилами написания качественного кода на Java. Дальнейшим развитием для выпускников программы является программа Java SE.

Что должен знать кандидат?

Кандидат должен отлично владеть синтаксисом C # (операции, операторы, var, as, is, for, foreach и т.д.), понимать .NET Framework, CLR. MSIL, уметь работать со сборками, GAC. Кандидат должен разбираться в типах данных, операциях со строками, ООП (классы, объекты, интерфейсы, свойства, инкапсуляция, наследование, полиморфизм), знать следующие понятия: делегаты и события, коллекции, дженерики, исключения, многопоточность, и тому подобное. Также будущий студент должен иметь базовые знания WinForms или WPF и уметь работать с XML и JSON.

Автоматизированное тестирование ПО - это процесс верификации программного обеспечения, при котором основные функции и шаги теста, такие как запуск, инициализация, выполнение, анализ и выдача результата, выполняются автоматически при помощи инструментов для автоматизированного тестирования. Automation QA специалисты являются очень востребованными на IT рынке Украины, поскольку их участие в проектах позволяет устранить большую часть рутинных операций и ускорить выполнение тестов.

Кандидат должен знать основные принципы ООП, владеть базовыми знаниями Java, понимать и хорошо ориентироваться в написании тесткейсов для мануального тестирования. Критическими являются навыки грамотного описания выявленной проблемы и умение мыслить логически. Наличие сертификата о завершении любой из программ BIONIC School, BIONIC University и BIONIC University Pro гарантирует 10% скидки на обучение.

Что будет знать и уметь выпускник?

Окончив курс, слушатели получат навыки работы в современной среде разработки программного обеспечения (IntelliJ IDEA, Maven, GIT), научатся писать автоматизированные тесты для WEB-UI и запускать их на сервере Continuous Integration, создавать репорты и ориентироваться в структуре DOM.
